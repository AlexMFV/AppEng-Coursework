Instructions / Shortcuts

Shortcuts

Although for some shortcuts the top buttons can be used, these are the shortcuts:

- Indent Textual Point: "Tab";

- De-Indent Textual Point: "Shift+Tab";

- Change Textual Point Level (1 to 7): "Ctrl+1"to"Ctrl+7"

- Force Save Document: "Ctrl+S"

Information

Without login the user can only save one document and it's stored locally on the user's device.

The login button redirects to the login page, when the user logins there are more features like (they appear after login):

- Save Multiple Documents with names

- Manage the documents (Edit, Rename, Delete)

- Edit/See the documents anywhere


Note: These shortcuts can be used on Windows with the "Ctrl" Key and on Mac with the Control "^" key

Note: The user can register on the "Login" page.
